# K Sound Galaxy

An interactive 3D music galaxy: every artist is a spiral nebula, every song a
star you can fly into, opening a particle "time mirror" of its album art with a
live 30-second preview. Built with React, Three.js, GSAP and a server-side
Gemini API for stylistic comparisons.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
